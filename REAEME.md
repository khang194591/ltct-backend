## Tech

Dillinger uses a number of open source projects to work properly:

- Sử dụng Type Script, Nodejs
- Supabase để lưu trữ database
- DBMS hệ quản trị cơ sở dữ liệu: PostgreSQL

## Cài đặt
```sh
cd ltct-backend
npm i
npm run dev
```
